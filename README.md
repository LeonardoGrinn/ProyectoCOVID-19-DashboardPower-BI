# ProyectoCOVID-19-DashboardPower-BI
Proyecto COVID-19 - Dashboard en Power BI

# Objetivo
Analizar información global de COVID-19 a partir de las tablas de casos confirmados, decesos y recuperados. El
proceso contempla importación, limpieza, despivoteo, combinación de tablas, creación de medidas DAX y visualización
final en un dashboard.

# Fuentes de datos utilizadas

Se utilizan las series globales de Johns Hopkins CSSE para casos confirmados, decesos y recuperados publicadas en
GitHub. Las tablas originales tienen columnas de ubicación y múltiples columnas de fecha, por lo que se transforman a
formato largo mediante Power Query.
